# Seattle Cycling Dashboard 🚲

An interactive data visualisation dashboard analysing bicycle 
and scooter traffic on the Fremont Bridge, Seattle (Oct 2012 – May 2026).

## Overview
This project explores over 119,000 hourly records of cycling 
activity, using Python to uncover trends by time of day, season, 
and direction of travel.

## Data Source
SOURCE: [Fremont Bridge Combined Bicycle and Scooter Counter](https://data.seattle.gov/Transportation/Fremont-Bridge-Combined-Bicycle-and-Scooter-Counte/65db-xm6k/about_data)  
City of Seattle Open Data Portal
